# GraphCMS тип Go

[Join our Slack](https://slack.graphcms.com)

This example demonstrates how to query from GraphCMS with Go.

## How to Use

### Download Manually

```bash
npx degit graphcms/graphcms-examples/with-golang with-golang
```

Install & Run:

```bash
cd with-golang
go mod download
go run main.go
```
